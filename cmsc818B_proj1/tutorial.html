Overview of the Assignment Problem

The assignment problem is a classical combinatorial problem where we are given a agents and t tasks to complete. Let's denote A as the set of agents and T as the set of tasks. Each possible pairing of an agent and task has an associated cost which we can describe as a mapping \(C: (A, T)\rightarrow \mathbb{R}\). Each agent can only complete one task and we need each task to be completed. The goal of the problem is to minimize the total cost to complete all of the tasks.
